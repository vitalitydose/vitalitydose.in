# Vitality Dose Website

Minimalist wellness website built with HTML & CSS.

## 🌿 Features
- Vitality Minimalist Design System
- Google Form Integration (Client Onboarding)
- Clean typography & luxury branding
- Responsive layout

## 🚀 Setup

1. Clone repo:
git clone https://github.com/yourusername/vitality-dose.git

2. Open project:
cd vitality-dose

3. Run:
Open index.html in browser

## 🔧 Customize
- Replace images in /assets
- Add your Google Form link
- Modify content in index.html
