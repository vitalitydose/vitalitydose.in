# Auth.md

Vitalitydose allows AI agents to interact with our public calculators and schedule clinical wellness consultations on behalf of users.

## Authentication Methods

For automated booking tools, agents must use OAuth 2.0 Client Credentials. 

## Registration

To register an agent, navigate to our [Free Consultation WhatsApp Portal](https://wa.me/917019890619) and request an API key. 

* **Issuer**: `https://vitalitydose.in`
* **Supported Scopes**: `read:tools`, `write:booking`
